# Viper/Eagle REP

Replacement PCB for Viper/Eagle V2/V3

## Status:
Initial prototypes ordered and tested. QMK and VIA added to main branch.

## Some features:
- QMK & VIA (soon)
- USB Mini
- STM32F072CB
- in-switich leds
- ISO and ANSI
- Underglow

# Viper/Eagle REP

Replacement PCB for Viper/Eagle V2/V3

## Status:
Fully tested and implemented. QMK and VIA added to main branch.

## Some features:
- QMK & VIA main repos
- USB C
- APM32F072
- in-switich leds
- ISO and ANSI
- Underglow true to the original PCB
